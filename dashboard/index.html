🚀 Quickstart Guide
  1. Install: pip install nexusre-mcp or uv pip install nexusre-mcp
  2. Configure: Run nexusre-mcp --install → auto-injects into Claude/Cursor
  3. Plugins: Run nexusre-mcp --install-plugins → copies plugins to IDA/Ghidra/x64dbg
  4. Open Binary: Load your target in IDA or Ghidra. The plugin starts automatically.
  5. First Command: Ask the AI: "Run full_analysis on the loaded binary"
  6. Pattern Hunt: Ask: "Find functions similar to this decryption routine"
  7. Vuln Scan: Ask: "Scan for vulnerabilities in the first 100 functions"
